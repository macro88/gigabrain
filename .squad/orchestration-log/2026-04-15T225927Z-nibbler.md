## Key Guardrails

### D1: Active Temporal Reads (BLOCKING)
Future-dated links (valid_from > today) must be excluded from default "active" views. Current implementation only checks `valid_until`, allowing 2099-dated links to appear as present-tense facts.

### D2: Graph Output Budgets (BLOCKING)
Hop cap of 10 alone does not protect against dense hub pages. Enforce max nodes, max edges, or max bytes. Traversal direction must be explicit in contract.

### D3: Contradiction Idempotency (BLOCKING)
- `extract_assertions` must only replace agent-generated assertions for target page
- Contradiction rows must deduplicate by stable fingerprint, not free-form text
- Rerunning `brain_check` must not create duplicates for same claim conflict

### D4: Gap Logging Dedup (BLOCKING)
Real dedupe key required (`query_hash` uniqueness). Anti-noise rules: repeat suppression + query sanity checks. Current schema has no UNIQUE constraint.

### D5: MCP Output Shape (BLOCKING)
MCP tools must return typed truth, not CLI stdout:
- `backlinks(..., _temporal, ...)` currently ignores temporal argument
- `timeline --json` returns `{ slug, entries }` not bare array
- `tags` mutates state but prints nothing
